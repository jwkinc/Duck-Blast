# Duck Blast Halloween PWA

Upload all files in this folder to a web host.

## GitHub Pages
1. Rename this folder's `index.html` stays as `index.html`.
2. Upload `index.html`, `manifest.json`, `service-worker.js`, `icon-192.png`, and `icon-512.png` to your GitHub repo.
3. Go to Settings → Pages.
4. Source: Deploy from branch.
5. Branch: main, folder: /root.
6. Open the GitHub Pages link on a phone.
7. Android: Chrome menu → Add to Home screen.
8. iPhone: Safari share button → Add to Home Screen.

The game will cache after first load and can run offline as a PWA.
